Project
==========
Python project to be able to get csv file and get aggregates and count


Language
===================
Choice of language --> PYTHON due to the ease of implememnting the assignment and ease of using the aggregate methods.

Usage
==================
1. Open any python ide.
2. python set environment path to folder that contains csv file
3. open jumo_loan_assignment.py
4. highlight the file and run the selection.
5. output.csv file will be generated in the project root.

Assumptions
==============
1. The csv file will always contain the given fields
2. The dates will always be in the same year.
3. The user will have python ide and python knowledge to be able to run this file and generate output file

plugins used
============
csv python library
